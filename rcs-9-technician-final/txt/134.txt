Purpose

The Connection even triggers an action when the agent finds an active network connection.
For the desktop device, enter the connection destination address.

For the mobile device, it triggers an action as soon as the device acquires a valid IP address on any
network interface (i.e.: WiFi, Activesync, GPRS/36+), and terminates the action when all the
