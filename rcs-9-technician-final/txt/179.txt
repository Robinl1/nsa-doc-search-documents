Step Action

2 From the Windows command prompt, run the following command to use the .pfx
ﬁle to sign Windows Phone applications:

where FiiePoth is the .pfx ﬁle path on the RC5 server

3 From the Windows command prompt, run the following command to use the .aetx
ﬁle as a Windows Phone application certificate:

rcs—db—config ——Sign—aetx—winphone EilePathlEﬁleName.aetx
where FiiePoth is the .aetx file path on the RC5 server
